## 调研工具概览

### 1. PyInstaller

#### 基本信息
- **官网**: https://pyinstaller.org/
- **GitHub Stars**: 11.5k+
- **活跃度**: 非常活跃
- **Python 支持**: 3.8 - 3.12
- **许可证**: GPL-2.0+ with special exception

#### 核心特性
| 特性 | 支持情况 |
|------|---------|
| 单文件模式 (--onefile) | ✅ |
| 单目录模式 (--onedir) | ✅ |
| 数据文件嵌入 | ✅ |
| 隐藏源码 | ✅ |
| UPX 压缩 | ✅ |
| 代码签名 | ✅ |

#### 优点
1. **社区成熟**: 最广泛使用的 Python 打包工具，文档完善，社区活跃
2. **易于使用**: 命令行简单易用，配置文件直观
3. **依赖处理**: 自动分析依赖，支持大部分 Python 库
4. **跨平台**: Windows、macOS、Linux 都支持
5. **隐藏源码**: 可以编译为二进制，保护源代码

#### 缺点
1. **文件较大**: 生成的可执行文件通常较大（50-200MB+）
2. **启动慢**: 单文件模式需要解压，启动时间较长
3. **杀毒软件误报**: 有时会被杀毒软件标记为可疑
4. **打包时间**: 大型项目打包可能需要几分钟

#### 本项目适用性评分
| 维度 | 评分 | 说明 |
|------|------|------|
| OpenCV/NumPy 支持 | ⭐⭐⭐⭐⭐ | 完全支持，社区有大量案例 |
| FastAPI 支持 | ⭐⭐⭐⭐⭐ | 完美支持，ASGI 应用可打包 |
| 前端资源嵌入 | ⭐⭐⭐⭐⭐ | 支持 --add-data 嵌入静态文件 |

## 🎯 推荐方案

### 主要推荐: **PyInstaller** ⭐⭐⭐⭐⭐

**推荐理由:**
1. **成熟稳定**: 社区最活跃，文档最完善，遇到问题容易找到解决方案
2. **完美支持**: OpenCV、NumPy、FastAPI 都完全支持，有大量成功案例
3. **前端嵌入**: 支持 `--add-data` 参数嵌入静态文件，非常适合本项目
4. **跨平台**: Windows 和 macOS 都支持良好
5. **易于使用**: 配置简单，学习成本低

**适用场景:**
- 需要快速开发、稳定运行的桌面应用
- 对文件大小和启动速度要求不是特别苛刻
- 需要良好的社区支持和文档

---

### 备选方案 1: **Nuitka** ⭐⭐⭐⭐

**适用场景:**
- 对执行性能有极高要求（需要 2-10 倍性能提升）
- 需要极高的源码保护（编译为二进制）
- 可以接受较长的编译时间和复杂的配置

**注意事项:**
- 编译时间可能很长（数十分钟到数小时）
- 需要测试 OpenCV 的 C 扩展兼容性
- 配置相对复杂，学习曲线较陡

---

### 备选方案 2: **cx_Freeze** ⭐⭐⭐⭐

**适用场景:**
- 希望生成的包比 PyInstaller 更小
- 对单文件模式没有硬性要求
- 愿意投入更多时间进行配置

**注意事项:**
- 不支持单文件模式，只能生成目录
- 社区和文档不如 PyInstaller 完善
- 某些依赖可能需要手动处理
